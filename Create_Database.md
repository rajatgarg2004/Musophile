Open MongoDB compass
Type in the URL section in New Connection the following
mongodb://localhost:27017
Click on Connect
Then on bottom right corner click on arrow key to open the console
Follow the steps given below to make the database and collections required for the above project. Please be cautious as it is case sensitive.

use Musophile; (Press Enter)
db.createCollection("songs"); (Press Enter)
db.createCollection("users"); (Press Enter)
